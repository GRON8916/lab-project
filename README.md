Описание проекта
Проект демонстрирует взаимодействие между веб-сервером на Node.js и базой данных PostgreSQL, запущенными в отдельных Docker-контейнерах.

Функциональность
Веб-сервер на Node.js с использованием Express

Подключение к PostgreSQL базе данных

REST API для выполнения SQL-запросов

Веб-интерфейс для тестирования функционала

Полная контейнеризация с помощью Docker Compose

Технологический стек
Backend: Node.js, Express

База данных: PostgreSQL

Контейнеризация: Docker, Docker Compose

Фронтенд: HTML, CSS, JavaScript

Локальная БД с другого предмета(файл 111).

Веб-приложение: http://localhost:3000
